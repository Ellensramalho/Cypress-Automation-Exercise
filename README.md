# 🚀 Cypress Automation Exercise

Projeto de automação de testes end-to-end utilizando **Cypress**, validação de fluxos web e boas práticas de QA.

---

## 🧪 Tecnologias

* Cypress
* JavaScript
* Node.js

---

## 📂 Estrutura

```
cypress/
 ├── e2e/
 ├── fixtures/
 ├── support/
```

## 💡 Objetivo

Praticar automação E2E e demonstrar habilidades em QA com Cypress.
