# 🚀 Cypress Automation Exercise

Projeto de automação de testes end-to-end utilizando **Cypress**, validação de fluxos web e boas práticas de QA.

---

## 🧪 Tecnologias

* Cypress
* JavaScript
* Node.js

---

## 📂 Estrutura

```
cypress/
 ├── e2e/
 ├── fixtures/
 ├── support/
```
---
O que foi testado
✔️ Navegação no site (Home → Products → categorias)
✔️ Seleção de produtos (categoria Women > Dress)
✔️ Visualização de detalhes do produto
✔️ Alteração de quantidade de itens
✔️ Adição de produto ao carrinho
✔️ Navegação até o carrinho e checkout
✔️ Preenchimento e envio de avaliação de produto
✔️ Fluxo de cadastro de usuário (sign up)
✔️ Preenchimento de formulário completo (dados pessoais, endereço, etc.)
✔️ Seleção de opções (gênero, data de nascimento, país)
✔️ Marcação de preferências (newsletter e ofertas)
✔️ Criação de conta
---
## 💡 Objetivo

Praticar automação E2E e demonstrar habilidades em QA com Cypress.
