🚀 Cypress Automation Exercise

Desenvolvi um projeto de automação de testes end-to-end utilizando **Cypress**, com foco na validação de fluxos reais de aplicações web e boas práticas de QA.

🧪 Tecnologias utilizadas:
• Cypress
• JavaScript
• Node.js

📂 Estrutura do projeto:
cypress/
├── e2e/
├── fixtures/
├── support/

🔍 O que foi testado:
✔️ Navegação no site (Home → Products → categorias).
✔️ Seleção de produtos (Women > Dress).
✔️ Visualização de detalhes do produto.
✔️ Alteração de quantidade de itens.
✔️ Adição de produtos ao carrinho.
✔️ Navegação até carrinho e checkout.
✔️ Envio de avaliação de produto.
✔️ Fluxo completo de cadastro de usuário (sign up).
✔️ Preenchimento de formulários (dados pessoais e endereço).
✔️ Interação com selects e checkboxes.
✔️ Criação de conta.

💡 Objetivo:
Praticar automação E2E e fortalecer minhas habilidades em QA, validando fluxos completos de usuário e interações com a interface.
