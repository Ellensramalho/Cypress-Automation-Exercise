# 🚀 Cypress Automation Exercise

Projeto de automação de testes end-to-end utilizando **Cypress**, validação de fluxos reais de aplicações web e boas práticas de QA.

---

## 🧪 Tecnologias utilizadas

- Cypress  
- JavaScript  
- Node.js  

---

## 📂 Estrutura do projeto


📂 Estrutura do projeto:
cypress/
├── e2e/
├── fixtures/
├── support/

---

## 🔍 O que foi testado

### 🛒 Fluxo de navegação e compra
- ✔️ Navegação no site (Home → Products → categorias).
- ✔️ Seleção de produtos (Women > Dress).
- ✔️ Visualização de detalhes do produto. 
- ✔️ Alteração de quantidade de itens.
- ✔️ Adição de produtos ao carrinho.
- ✔️ Navegação até carrinho e checkout.  

### ⭐ Interações do usuário
- ✔️ Envio de avaliação de produto.

### 👤 Cadastro de usuário
- ✔️ Fluxo completo de cadastro (sign up).
- ✔️ Preenchimento de formulário (dados pessoais e endereço).
- ✔️ Seleção de opções (gênero, data, país).
- ✔️ Interação com selects e checkboxes.
- ✔️ Criação de conta.

---

## 💡 Objetivo

Praticar automação E2E e fortalecer habilidades em QA, validando fluxos completos de usuário e interações com a interface.
